# SeaBattleStartPositionGenerator
This program on C++ will give you a random start position in game "Sea Battle".
You will get a square 10x10 which contains a 10 ships with differet lengths:
1 four-deck ship;
2 three-deck ships;
3 two-deck ships;
4 one-deck ships.
